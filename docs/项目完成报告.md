## 使用指南

### 安装
```bash
cd E:\AI_Projects\NovelExtractor
pip install -e .
```

### 配置环境变量
```bash
export DEEPSEEK_API_KEY="your-api-key"
export DEEPSEEK_BASE_URL="https://api.deepseek.com"
export DEEPSEEK_MODEL="deepseek-chat"
```

### 查看执行计划
```bash
novel-extractor plan --config config/novel_extractor.mvp.yaml
```

### 执行提取（前台）
```bash
novel-extractor run --config config/novel_extractor.mvp.yaml
```

### 中断后恢复
```bash
novel-extractor resume --config config/novel_extractor.mvp.yaml
```

### 查看状态
```bash
novel-extractor status --config config/novel_extractor.mvp.yaml
```

### 重置窗口
```bash
novel-extractor reset-window --config config/novel_extractor.mvp.yaml --window 5-9 --group resource_group
```

## 成本估算

### MVP 验证成本（max_windows: 3）
- **窗口数**: 3
- **每窗口 tokens**: ~25,000（章节 20k + 模板 5k）
- **模板组数**: 4
- **总 tokens**: 3 × 4 × 25,000 = 300,000 tokens
- **估算成本**: $0.6-1.2（DeepSeek 价格）

### 全书处理成本（无缓存）
- **全书章节**: ~2000 章
- **窗口数**: ~500
- **总 tokens**: 500 × 4 × 25,000 = 50,000,000 tokens
- **估算成本**: $100-200

### 全书处理成本（有缓存优化）
- **缓存命中率**: ~60%（模板部分）
- **实际 tokens**: ~20,000,000 tokens
- **估算成本**: $40-80

## 已知限制

### 技术限制
1. **前台执行**: 必须保持控制台打开
2. **单线程**: 无并发处理
3. **简单路由**: 字符级匹配可能误触发
4. **无向量化**: 不支持语义搜索去重

### 配置限制
1. **固定窗口**: 大小和步长在配置中固定
2. **模板数量**: MVP 只支持 5 个模板
3. **单一小说**: 配置文件绑定一本小说

### 输出限制
1. **Markdown 格式**: 模型必须严格遵守格式
2. **手工审查**: 需要人工验证输出质量
3. **无自动合并**: 重复条目需要人工去重
