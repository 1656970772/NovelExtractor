# 工具调用协议复刻 opencode 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 NovelExtractor 的工具调用流程改成和 opencode 同型的“通用 ToolDefinition + 协议 adapter 降级 + provider-native request body + provider-native tool call 解析 + 本地 schema 校验 + 完整请求快照日志”架构。

**Architecture:** 工具定义只保留一份权威结构：`name / description / inputSchema / outputSchema`。LLM 客户端不再把工具提前固定成 OpenAI Chat 的 `{ type, function }` 形状，而是在发送前按 `apiFormat` 降级成 OpenAI Chat、OpenAI Responses、Anthropic Messages、Gemini GenerateContent、Bedrock Converse 各自协议。日志不再打印拍扁的 `name + parameters` 工具表，而是记录发送前编译好的 provider-native body，普通简洁日志继续脱敏和摘要化。

**Tech Stack:** TypeScript, pnpm workspace, Vitest, Electron main process, existing `@novel-extractor/llm`, `@novel-extractor/tools`, `@novel-extractor/config`, `@novel-extractor/desktop`.

---

## 背景结论

opencode 的关键做法不是把工具说明塞进普通 prompt，而是：

- `Tool.make` 生成通用 `ToolDefinition`：`name / description / inputSchema / outputSchema`。
- session runner 把 `tools: ToolDefinition[]` 放进通用 `LLMRequest`。
- protocol adapter 在发送前降级：
  - OpenAI Chat: `tools: [{ type: "function", function: { name, description, parameters } }]`
  - OpenAI Responses: `tools: [{ type: "function", name, description, parameters }]`
  - Anthropic Messages: `tools: [{ name, description, input_schema }]`
  - Gemini: `tools: [{ functionDeclarations: [{ name, description, parameters }] }]`
  - Bedrock Converse: `toolConfig.tools: [{ toolSpec: { name, description, inputSchema: { json } } }]`
- 工具参数由模型在响应协议的工具调用字段里返回；OpenAI 风格通常是 JSON 字符串，内部解析后再执行工具。
- 数组参数只需要在 JSON Schema 中写成 `type: "array"` 和 `items`，不需要用 prompt 约定逗号字符串。

当前 NovelExtractor 的问题：

- `packages/llm/src/openAiCompatibleClient.ts` 的公开 `ToolSchema` 已经固定为 OpenAI Chat 的 `{ type: "function", function: ... }`。
- `apps/desktop/src/main/windowRunService.ts` 的 `toLlmToolSchema()` 也提前生成 OpenAI Chat shape。
- `apps/desktop/src/main/taskTextLogger.ts` 的 `serializeToolsForTaskLog()` 把工具日志拍扁成 `{ name, parameters }`，看不到 `type/function/description/input_schema` 等真实请求结构。
- `packages/config/src/schema.ts` 的 `ProviderApiFormat` 目前只有 `"openai_chat" | "openai_responses"`，没有 Anthropic/Gemini/Bedrock 原生协议。

硬要求：

- 每个 adapter task 的实现步骤前，先执行文档里的 `Get-Content` 参考源码命令。
- 优先搬运 opencode 的数据流和字段映射，再按本项目已有 `ChatCompletionMessage`、`ToolCall`、`ProviderApiFormat` 类型做最小适配。
- 如果 opencode 已有某个协议的消息降级、tool call 回放、schema 投影、响应解析逻辑，不要重新设计名称不同但行为类似的新机制。
- 只有 Bedrock AWS SigV4 是当前项目缺少的外部认证能力，允许先明确阻断未签名真实请求，但 body 编译仍要复刻 opencode 的 Converse shape。

执行约束补充：

- 本计划只扩展 `ProviderApiFormat` 和 `authScheme`；`ProviderKind` 继续保持 `"openai-compatible"`。Anthropic/Gemini/Bedrock 的协议差异由 `apiFormat` 和 adapter 承载，不在本轮引入新的 provider kind，避免连带改动桌面端 provider UI、domain 类型和既有 provider preset 断言。
- OpenAI 风格 tool call arguments 的编码/解析必须集中复用共享 helper，并保留当前客户端已有的兼容行为：字符串 arguments 原样发送；解析响应时先尝试 `JSON.parse`，失败后继续支持 `"{}" + JSON object` 前缀容错。
- 非 OpenAI 协议必须显式处理 `system` 消息：Anthropic 用顶层 `system`；Gemini 用 `systemInstruction`；Bedrock Converse 用顶层 `system`。不要把 system prompt 静默降级成普通 user message。
- provider-native request body 日志必须继续走窗口原文替换和密钥脱敏边界；不能因为记录完整 body 而把 `## 当前窗口文本` 原文写入任务日志。
- 执行 Task 8 和 Task 10 时要把“编译 provider-native body”和“记录 provider-native body”视为同一条请求管线的两个阶段：Task 8 先完成 adapter 编译和发送；Task 10 只增加 `onRequestPrepared` 快照回调和日志落点，不重新引入旧的 OpenAI Chat 工具结构。
- 执行 Task 11 时不要绕过当前 `windowRunService.ts` 的 pre-execution recoverable error 分类链。schema 校验失败必须通过 `ToolExecutionError("invalid args: ...", "INVALID_ARGUMENTS")` 进入现有 `classifyToolExecutionError()`，这样日志中的 `reason` 仍会是 `tool_schema_invalid_arguments`，重复错误统计和修正轮次也继续生效。

- [ ] **Step 3: 创建通用工具定义文件**

Create `packages/llm/src/toolDefinition.ts`:

```ts
export type JsonSchemaValue =
  | string
  | number
  | boolean
  | null
  | JsonSchemaValue[]
  | { [key: string]: JsonSchemaValue };

export type JsonObject = { [key: string]: JsonSchemaValue };

export interface ToolDefinition {
  name: string;
  description: string;
  inputSchema: JsonObject;
  outputSchema?: JsonObject;
}

export interface ProviderToolSource {
  name: string;
  description: string;
  parameters: unknown;
}

export function isJsonObject(value: unknown): value is JsonObject {
  return value !== null && typeof value === "object" && !Array.isArray(value);
}

export function stableJsonValue(value: unknown): unknown {
  if (Array.isArray(value)) {
    return value.map(stableJsonValue);
  }

  if (isJsonObject(value)) {
    return Object.fromEntries(
      Object.keys(value)
        .sort()
        .map((key) => [key, stableJsonValue(value[key])])
    );
  }

  return value;
}

export function normalizeInputSchema(value: unknown): JsonObject {
  if (isJsonObject(value)) {
    return stableJsonValue(value) as JsonObject;
  }

  return { type: "object" };
}

export function toToolDefinition(tool: ProviderToolSource): ToolDefinition {
  return {
    name: tool.name,
    description: tool.description,
    inputSchema: normalizeInputSchema(tool.parameters)
  };
}

export function encodeToolArguments(value: unknown): string {
  if (typeof value === "string") {
    return value;
  }

  return JSON.stringify(stableJsonValue(value)) ?? "";
}
```

```ts
import type { JsonObject } from "../toolDefinition";
import { isJsonObject, stableJsonValue } from "../toolDefinition";

export type ToolSchemaProjectionTarget = "openai" | "anthropic" | "gemini" | "bedrock";

function removeNullSchemas(value: unknown): unknown {
  if (Array.isArray(value)) {
    return value.map(removeNullSchemas);
  }

  if (!isJsonObject(value)) {
    return value;
  }

  const fields = Object.fromEntries(
    Object.entries(value)
      .filter(([key]) => key !== "anyOf")
      .map(([key, item]) => [key, removeNullSchemas(item)])
  );

  if (!Array.isArray(value.anyOf)) {
    return fields;
  }

  const variants = value.anyOf
    .filter((variant) => !isJsonObject(variant) || variant.type !== "null")
    .map(removeNullSchemas);

  if (variants.length === 1 && isJsonObject(variants[0])) {
    return { ...fields, ...variants[0] };
  }

  return { ...fields, anyOf: variants };
}

function stripKeys(value: unknown, blockedKeys: ReadonlySet<string>): unknown {
  if (Array.isArray(value)) {
    return value.map((item) => stripKeys(item, blockedKeys));
  }

  if (!isJsonObject(value)) {
    return value;
  }

  return Object.fromEntries(
    Object.entries(value)
      .filter(([key]) => !blockedKeys.has(key))
      .map(([key, item]) => [key, stripKeys(item, blockedKeys)])
  );
}

function openAiProjection(schema: JsonObject): JsonObject {
  const normalized = removeNullSchemas({ ...schema, type: "object" });
  return (isJsonObject(normalized) ? stableJsonValue(normalized) : { type: "object" }) as JsonObject;
}

function geminiProjection(schema: JsonObject): JsonObject {
  const projected = stripKeys(schema, new Set(["additionalProperties", "$schema", "$defs", "definitions"]));
  return (isJsonObject(projected) ? stableJsonValue(projected) : { type: "object" }) as JsonObject;
}

export function projectToolSchema(target: ToolSchemaProjectionTarget, schema: JsonObject): JsonObject {
  switch (target) {
    case "openai":
      return openAiProjection(schema);
    case "gemini":
      return geminiProjection(schema);
    case "anthropic":
    case "bedrock":
      return stableJsonValue(schema) as JsonObject;
  }
}
```

```ts
import type { ChatCompletionMessage, ToolCall } from "../openAiCompatibleClient";
import { encodeToolArguments } from "../toolDefinition";
import { parseProtocolToolArguments } from "./shared";
import { projectToolSchema } from "./toolSchemaProjection";
import type { BuildProtocolBodyInput, LlmProtocolAdapter } from "./types";

function asRecord(value: unknown): Record<string, unknown> | undefined {
  return value !== null && typeof value === "object" && !Array.isArray(value)
    ? (value as Record<string, unknown>)
    : undefined;
}

function serializeToolCall(toolCall: NonNullable<Extract<ChatCompletionMessage, { role: "assistant" }>["toolCalls"]>[number]) {
  return {
    id: toolCall.id,
    type: "function" as const,
    function: {
      name: toolCall.name,
      arguments: encodeToolArguments(toolCall.arguments)
    }
  };
}

function serializeMessage(message: ChatCompletionMessage): unknown {
  if (message.role === "assistant") {
    return {
      role: "assistant",
      content: message.content,
      ...(message.toolCalls?.length ? { tool_calls: message.toolCalls.map(serializeToolCall) } : {})
    };
  }

  if (message.role === "tool") {
    return {
      role: "tool",
      tool_call_id: message.toolCallId,
      ...(message.name ? { name: message.name } : {}),
      content: message.content
    };
  }

  return message;
}

function lowerTools(input: BuildProtocolBodyInput): unknown[] | undefined {
  if (input.tools.length === 0) {
    return undefined;
  }

  return input.tools.map((tool) => ({
    type: "function" as const,
    function: {
      name: tool.name,
      description: tool.description,
      parameters: projectToolSchema("openai", tool.inputSchema)
    }
  }));
}

function parseToolCalls(value: unknown): ToolCall[] {
  if (!Array.isArray(value)) {
    return [];
  }

  return value.flatMap((item) => {
    const record = asRecord(item);
    if (record?.type !== "function") {
      return [];
    }

    const fn = asRecord(record.function);
    if (typeof fn?.name !== "string" || fn.name.trim() === "") {
      return [];
    }

    return [{
      id: typeof record.id === "string" ? record.id : "",
      name: fn.name,
      arguments: parseProtocolToolArguments(fn.arguments)
    }];
  });
}

export const openAiChatAdapter: LlmProtocolAdapter = {
  apiFormat: "openai_chat",
  path: () => "/chat/completions",
  buildBody(input) {
    return {
      model: input.modelId,
      messages: input.messages.map(serializeMessage),
      ...(lowerTools(input) ? { tools: lowerTools(input) } : {})
    };
  },
  parseResponse(responseBody) {
    const body = asRecord(responseBody);
    const choices = Array.isArray(body?.choices) ? body.choices : [];
    const message = asRecord(asRecord(choices[0])?.message);
    return {
      content: typeof message?.content === "string" ? message.content : "",
      usage: body?.usage,
      toolCalls: parseToolCalls(message?.tool_calls)
    };
  }
};
```

```ts
import type { ChatCompletionMessage, ToolCall } from "../openAiCompatibleClient";
import { encodeToolArguments } from "../toolDefinition";
import { parseProtocolToolArguments } from "./shared";
import { projectToolSchema } from "./toolSchemaProjection";
import type { BuildProtocolBodyInput, LlmProtocolAdapter } from "./types";

function asRecord(value: unknown): Record<string, unknown> | undefined {
  return value !== null && typeof value === "object" && !Array.isArray(value)
    ? (value as Record<string, unknown>)
    : undefined;
}

function serializeMessage(message: ChatCompletionMessage): unknown[] {
  if (message.role === "assistant") {
    return [
      ...(message.content ? [{ role: "assistant" as const, content: message.content }] : []),
      ...(message.toolCalls?.map((toolCall) => ({
        type: "function_call" as const,
        call_id: toolCall.id,
        name: toolCall.name,
        arguments: encodeToolArguments(toolCall.arguments)
      })) ?? [])
    ];
  }

  if (message.role === "tool") {
    return [
      {
        type: "function_call_output" as const,
        call_id: message.toolCallId,
        output: message.content
      }
    ];
  }

  return [message];
}

function lowerTools(input: BuildProtocolBodyInput): unknown[] | undefined {
  if (input.tools.length === 0) {
    return undefined;
  }

  return input.tools.map((tool) => ({
    type: "function" as const,
    name: tool.name,
    description: tool.description,
    parameters: projectToolSchema("openai", tool.inputSchema),
    strict: false
  }));
}

function parseToolCalls(output: unknown): ToolCall[] {
  if (!Array.isArray(output)) {
    return [];
  }

  return output.flatMap((item) => {
    const record = asRecord(item);
    if (record?.type !== "function_call" || typeof record.name !== "string") {
      return [];
    }

    return [{
      id: typeof record.call_id === "string" ? record.call_id : "",
      name: record.name,
      arguments: parseProtocolToolArguments(record.arguments)
    }];
  });
}

function parseOutputText(body: Record<string, unknown> | undefined): string {
  if (typeof body?.output_text === "string") {
    return body.output_text;
  }

  if (!Array.isArray(body?.output)) {
    return "";
  }

  return body.output
    .flatMap((outputItem) => {
      const outputRecord = asRecord(outputItem);
      const content = outputRecord?.content;
      if (!Array.isArray(content)) {
        return [];
      }

      return content.flatMap((contentItem) => {
        const contentRecord = asRecord(contentItem);
        return typeof contentRecord?.text === "string" ? [contentRecord.text] : [];
      });
    })
    .join("");
}

export const openAiResponsesAdapter: LlmProtocolAdapter = {
  apiFormat: "openai_responses",
  path: () => "/responses",
  buildBody(input) {
    return {
      model: input.modelId,
      input: input.messages.flatMap(serializeMessage),
      ...(lowerTools(input) ? { tools: lowerTools(input) } : {})
    };
  },
  parseResponse(responseBody) {
    const body = asRecord(responseBody);
    return {
      content: parseOutputText(body),
      usage: body?.usage,
      toolCalls: parseToolCalls(body?.output)
    };
  }
};
```

```ts
import type { ChatCompletionMessage, ToolCall } from "../openAiCompatibleClient";
import { projectToolSchema } from "./toolSchemaProjection";
import type { BuildProtocolBodyInput, LlmProtocolAdapter } from "./types";

function asRecord(value: unknown): Record<string, unknown> | undefined {
  return value !== null && typeof value === "object" && !Array.isArray(value)
    ? (value as Record<string, unknown>)
    : undefined;
}

function textBlock(content: string): Array<{ type: "text"; text: string }> {
  return [{ type: "text", text: content }];
}

type SystemMessage = ChatCompletionMessage & { role: "system"; content: string };
type NonSystemMessage = ChatCompletionMessage & { role: "user" | "assistant" | "tool" };

function isSystemMessage(message: ChatCompletionMessage): message is SystemMessage {
  return message.role === "system";
}

function isNonSystemMessage(message: ChatCompletionMessage): message is NonSystemMessage {
  return message.role !== "system";
}

function serializeMessage(message: NonSystemMessage): unknown {
  if (message.role === "tool") {
    return {
      role: "user",
      content: [
        {
          type: "tool_result",
          tool_use_id: message.toolCallId,
          content: message.content
        }
      ]
    };
  }

  if (message.role === "assistant") {
    return {
      role: "assistant",
      content: [
        ...(message.content ? textBlock(message.content) : []),
        ...(message.toolCalls?.map((toolCall) => ({
          type: "tool_use" as const,
          id: toolCall.id,
          name: toolCall.name,
          input: toolCall.arguments
        })) ?? [])
      ]
    };
  }

  return {
    role: message.role,
    content: textBlock(message.content)
  };
}

function lowerMessages(messages: ChatCompletionMessage[]): { system?: string; messages: unknown[] } {
  const system = messages
    .filter(isSystemMessage)
    .map((message) => message.content)
    .join("\n\n");
  const loweredMessages = messages
    .filter(isNonSystemMessage)
    .map(serializeMessage);

  return {
    ...(system ? { system } : {}),
    messages: loweredMessages
  };
}

function lowerTools(input: BuildProtocolBodyInput): unknown[] | undefined {
  if (input.tools.length === 0) {
    return undefined;
  }

  return input.tools.map((tool) => ({
    name: tool.name,
    description: tool.description,
    input_schema: projectToolSchema("anthropic", tool.inputSchema)
  }));
}

function parseContentBlocks(content: unknown): { text: string[]; toolCalls: ToolCall[] } {
  if (!Array.isArray(content)) {
    return { text: [], toolCalls: [] };
  }

  const text: string[] = [];
  const toolCalls: ToolCall[] = [];

  for (const item of content) {
    const record = asRecord(item);
    if (record?.type === "text" && typeof record.text === "string") {
      text.push(record.text);
    }
    if (record?.type === "tool_use" && typeof record.name === "string") {
      toolCalls.push({
        id: typeof record.id === "string" ? record.id : "",
        name: record.name,
        arguments: record.input
      });
    }
  }

  return { text, toolCalls };
}

export const anthropicMessagesAdapter: LlmProtocolAdapter = {
  apiFormat: "anthropic_messages",
  path: () => "/messages",
  buildBody(input) {
    const lowered = lowerMessages(input.messages);
    return {
      model: input.modelId,
      ...(lowered.system ? { system: lowered.system } : {}),
      messages: lowered.messages,
      ...(lowerTools(input) ? { tools: lowerTools(input) } : {})
    };
  },
  parseResponse(responseBody) {
    const body = asRecord(responseBody);
    const parsed = parseContentBlocks(body?.content);
    return {
      content: parsed.text.join(""),
      usage: body?.usage,
      toolCalls: parsed.toolCalls
    };
  }
};
```

```ts
  return value !== null && typeof value === "object" && !Array.isArray(value)
    ? (value as Record<string, unknown>)
    : undefined;
}

type SystemMessage = ChatCompletionMessage & { role: "system"; content: string };
type NonSystemMessage = ChatCompletionMessage & { role: "user" | "assistant" | "tool" };

function isSystemMessage(message: ChatCompletionMessage): message is SystemMessage {
  return message.role === "system";
}

function isNonSystemMessage(message: ChatCompletionMessage): message is NonSystemMessage {
  return message.role !== "system";
}

function serializeMessage(message: NonSystemMessage): unknown {
  if (message.role === "tool") {
    return {
      role: "user",
      parts: [
        {
          functionResponse: {
            name: message.name ?? "tool_result",
            response: { content: message.content }
          }
        }
      ]
    };
  }

  if (message.role === "assistant") {
    return {
      role: "model",
      parts: [
        ...(message.content ? [{ text: message.content }] : []),
        ...(message.toolCalls?.map((toolCall) => ({
          functionCall: {
            name: toolCall.name,
            args: toolCall.arguments
          }
        })) ?? [])
      ]
    };
  }

  return {
    role: "user",
    parts: [{ text: message.content }]
  };
}

function lowerMessages(messages: ChatCompletionMessage[]): {
  systemInstruction?: { parts: Array<{ text: string }> };
  contents: unknown[];
} {
  const systemText = messages
    .filter(isSystemMessage)
    .map((message) => message.content)
    .join("\n\n");
  const contents = messages
    .filter(isNonSystemMessage)
    .map(serializeMessage);

  return {
    ...(systemText ? { systemInstruction: { parts: [{ text: systemText }] } } : {}),
    contents
  };
}

function lowerTools(input: BuildProtocolBodyInput): unknown[] | undefined {
  if (input.tools.length === 0) {
    return undefined;
  }

  return [
    {
      functionDeclarations: input.tools.map((tool) => ({
        name: tool.name,
        description: tool.description,
        parameters: projectToolSchema("gemini", tool.inputSchema)
      }))
    }
  ];
}

function parseToolCalls(parts: unknown): ToolCall[] {
  if (!Array.isArray(parts)) {
    return [];
  }

  let index = 0;
  return parts.flatMap((part) => {
    const functionCall = asRecord(asRecord(part)?.functionCall);
    if (typeof functionCall?.name !== "string") {
      return [];
    }

    index += 1;
    return [{
      id: `gemini-call-${index}`,
      name: functionCall.name,
      arguments: functionCall.args ?? {}
    }];
  });
}

function parseText(parts: unknown): string {
  if (!Array.isArray(parts)) {
    return "";
  }

  return parts
    .map((part) => asRecord(part)?.text)
    .filter((text): text is string => typeof text === "string")
    .join("");
}

export const geminiGenerateContentAdapter: LlmProtocolAdapter = {
  apiFormat: "gemini_generate_content",
  path: ({ modelId }) => `/models/${encodeURIComponent(modelId)}:generateContent`,
  buildBody(input) {
    const lowered = lowerMessages(input.messages);
    return {
      ...(lowered.systemInstruction ? { systemInstruction: lowered.systemInstruction } : {}),
      contents: lowered.contents,
      ...(lowerTools(input) ? { tools: lowerTools(input) } : {})
    };
  },
  parseResponse(responseBody) {
    const body = asRecord(responseBody);
    const candidates = Array.isArray(body?.candidates) ? body.candidates : [];
    const parts = asRecord(asRecord(candidates[0])?.content)?.parts;
    return {
      content: parseText(parts),
      usage: body?.usageMetadata,
      toolCalls: parseToolCalls(parts)
    };
  }
};
```

```ts
export interface ToolValidationError {
  path: string;
  message: string;
}

export function validateToolArguments(schema: unknown, value: unknown): ToolValidationError[] {
  return validateJsonSchemaValue(schema, value, "$");
}

function validateJsonSchemaValue(schema: unknown, value: unknown, path: string): ToolValidationError[] {
  if (!schema || typeof schema !== "object" || Array.isArray(schema)) {
    return [];
  }

  const record = schema as Record<string, unknown>;
  const type = record.type;

  if (type === "object") {
    if (value === null || typeof value !== "object" || Array.isArray(value)) {
      return [{ path, message: "必须是对象" }];
    }

    const valueRecord = value as Record<string, unknown>;
    const required = Array.isArray(record.required) ? record.required.filter((item): item is string => typeof item === "string") : [];
    const missing = required.flatMap((key) => key in valueRecord ? [] : [{ path: `${path}.${key}`, message: "缺少必填字段" }]);
    const properties = record.properties && typeof record.properties === "object" && !Array.isArray(record.properties)
      ? (record.properties as Record<string, unknown>)
      : {};
    const nested = Object.entries(properties).flatMap(([key, childSchema]) =>
      key in valueRecord ? validateJsonSchemaValue(childSchema, valueRecord[key], `${path}.${key}`) : []
    );
    const extra = record.additionalProperties === false
      ? Object.keys(valueRecord)
          .filter((key) => !(key in properties))
          .map((key) => ({ path: `${path}.${key}`, message: "不允许额外字段" }))
      : [];
    return [...missing, ...extra, ...nested];
  }

  if (type === "array") {
    if (!Array.isArray(value)) {
      return [{ path, message: "必须是数组" }];
    }

    const minItems = typeof record.minItems === "number" ? record.minItems : undefined;
    const lengthErrors = minItems !== undefined && value.length < minItems
      ? [{ path, message: `数组长度不能少于 ${minItems}` }]
      : [];
    const itemSchema = record.items;
    const itemErrors = value.flatMap((item, index) => validateJsonSchemaValue(itemSchema, item, `${path}[${index}]`));
    return [...lengthErrors, ...itemErrors];
  }

  if (type === "string" && typeof value !== "string") {
    return [{ path, message: "必须是字符串" }];
  }

  if (type === "integer" && (!Number.isInteger(value))) {
    return [{ path, message: "必须是整数" }];
  }

  if (type === "number" && (typeof value !== "number" || !Number.isFinite(value))) {
    return [{ path, message: "必须是数字" }];
  }

  if ((type === "integer" || type === "number") && typeof value === "number" && Number.isFinite(value)) {
    const minimum = typeof record.minimum === "number" ? record.minimum : undefined;
    const maximum = typeof record.maximum === "number" ? record.maximum : undefined;
    return [
      ...(minimum !== undefined && value < minimum ? [{ path, message: `不能小于 ${minimum}` }] : []),
      ...(maximum !== undefined && value > maximum ? [{ path, message: `不能大于 ${maximum}` }] : [])
    ];
  }

  if (type === "boolean" && typeof value !== "boolean") {
    return [{ path, message: "必须是布尔值" }];
  }

  return [];
}
```
